# ThinkTodo

> Think is series of software that supports manual by keyboard.

Create, check and complete todo items, all by keyboard.

# Requirement

- electron
- react
- node

# Feature

All the features support by shortcuts or manual by keyboard.

Key features
- create todo item
- update todo item
- complete todo item

Better features
- tab
- archive
- deadline
